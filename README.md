# Exploratory Data Analysis (EDA) Project

## Objective
Analyze a sales dataset to identify patterns, trends, and business insights.

## Dataset Information
- Dataset: Sales Dataset
- Records: 1000
- Columns: 14

## Tools Used
- Microsoft Excel
- Google Docs
- GitHub

## Analysis Performed
1. Region-wise Sales Analysis
2. Product Category Distribution Analysis
3. Data Visualization using Charts

## Key Findings
- North region recorded the highest sales.
- South region recorded the lowest sales.
- Clothing category had the highest count.
- Product distribution was analyzed using pie charts.

## Files Included
- EDA_Report.pdf
- README.md

## Conclusion
The analysis provided useful insights into sales performance and product category trends.
